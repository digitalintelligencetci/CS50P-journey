## 💡 Key Concepts

- Everything in a parenthesis are the arguments to the function.
- concatenation
- comments are necessary for massive code
- (*) objects mean the print function can take any number of objects
- sep=' ' (separator)
- end='\n' - end the line and move to a new line.
- parameters
- positional parameters (prints in sequential order)
- escaping
- f (format)
- name.strip() - removes whitespace from string
- capitalize / name.title()
- string methods can be on one line (cuts down on the lines of code also called method chaining)
- integers
- float (number with a decimal point value)
- interactive move
- type converter: int, float
- input is a function
- round function, decimals and places
- define (def)
    - () no inputs or arguments passed
    - colon means stay tuned for indentations. Everything indented (beneath the line of code) is apart of the function.
